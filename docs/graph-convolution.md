# 图卷积神经网络 (GCN) 核心笔记

相比于在规则网格上操作的经典 CNN（如 MobileNet 等），图卷积的核心目的是处理不规则的图拓扑结构数据，提取节点间的深层空间依赖特征。这对于我们后续探索复杂多变量场景下的多尺度特征融合，具有极高的参考价值。

## 1. 核心数学表达

图卷积的每一次操作，本质上是在做**消息传递与聚合**。Kipf 和 Welling 提出的经典前向传播公式如下：

$$H^{(l+1)} = \sigma(\tilde{D}^{-\frac{1}{2}}\tilde{A}\tilde{D}^{-\frac{1}{2}}H^{(l)}W^{(l)})$$

**公式参数解析：**
* $H^{(l)}$：第 $l$ 层的节点特征矩阵。
* $\tilde{A}$：加入了自环的邻接矩阵（$\tilde{A} = A + I$），确保节点更新时保留自身历史信息。
* $\tilde{D}$：$\tilde{A}$ 的度矩阵，配合进行对称归一化，防止特征值爆炸。
* $W^{(l)}$：当前层的可学习权重矩阵。
* $\sigma$：非线性激活函数（例如我们轻量化模型中常用的 GELU 或 Hard-GELU）。

## 2. 核心优势

GCN 打破了固定卷积核感受野的物理限制，能够显式地建模实体之间的依赖关系，直接在特征空间中建立跨越物理距离的长程语义关联。
